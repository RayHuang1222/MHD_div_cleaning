# MHDtest
Final term project for the Computational Astrophysics class in NTU.

The following Riemann solvers have been implemented for hydrodynamics problems. 

 1.Roe Solver
 2.HLL Solver
 3.HLLC Solver

The following Riemann solvers have been implemented for magnetohydroidynamics problems.
